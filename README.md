# LrcCaptionGenerator

将 LRC 歌词文件转换为带动画效果的透明背景字幕视频，无需安装任何软件，直接在浏览器中运行。

## 功能特性

- 解析标准 LRC 歌词文件，自动同步字幕时间轴
- 10 种入场动画、3 种待机动画、6 种退场动画，随机组合
- 支持自定义背景（纯色或图片）、字体颜色与描边
- 实时预览，可同步播放音频
- 导出透明背景视频：WebM（VP9）或 MOV（ProRes 4444）
- 相同随机种子可复现完全一致的动画效果
- 纯前端，无需服务器，可作为静态页面部署

## 快速开始

### 环境要求

- Node.js 18+
- 现代浏览器（Chrome / Edge 推荐，需支持 MediaRecorder API）

### 安装与运行

```bash
cd LrcCaptionGenerator
npm install
npm run dev
```

浏览器访问 `http://localhost:5173` 即可使用。

## 使用说明

### 第一步：输入歌词

在左侧面板的文本框中粘贴 LRC 格式歌词，或点击「上传 LRC 文件」按钮导入文件。

LRC 格式示例：

```
[00:12.34]第一句歌词
[00:16.78]第二句歌词
[00:21.00]第三句歌词
```

### 第二步：配置参数

| 参数 | 说明 |
|------|------|
| **随机种子** | 控制动画随机性，相同种子产生相同效果；留空则每次随机 |
| **分辨率** | 1920×1080（横屏）、1080×1920（竖屏）、1080×1080（方形）、1280×720 |
| **背景** | 纯色或图片；图片可调节亮度、对比度、饱和度 |
| **文字颜色** | 字体填充色 |
| **描边** | 描边宽度（0 为无描边）与描边颜色 |

### 第三步：生成预览

点击「生成预览」按钮，右侧画布将显示动画效果。使用播放控件预览完整时间轴。

可选：上传音频文件以同步播放。

### 第四步：导出

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| **PNG** | 导出当前帧静态图片 | 封面、缩略图 |
| **WebM** | VP9 编码，含透明通道，文件较小 | Premiere、DaVinci Resolve 等现代剪辑软件 |
| **MOV** | ProRes 4444，含透明通道，兼容性最佳 | Final Cut Pro、旧版剪辑软件 |

> MOV 导出首次使用时需加载约 30MB 的 ffmpeg.wasm，请耐心等待。

## 动画效果一览

### 入场（10 种）

| 效果名 | 描述 |
|--------|------|
| typewriter | 逐字打字机效果 |
| slideLeft / slideRight | 从左/右侧滑入 |
| slideUp / slideDown | 从下/上方滑入 |
| scalePop | 弹性缩放弹出 |
| scatter | 从随机位置汇聚 |
| flipX | 水平翻转入场 |
| blurFade | 模糊渐显 |
| wave | 波浪式依次出现 |

### 待机（3 种）

| 效果名 | 描述 |
|--------|------|
| float | 整行上下浮动 |
| charJitter | 各字符微抖动 |
| breathe | 整行呼吸缩放 |

### 退场（6 种）

| 效果名 | 描述 |
|--------|------|
| fadeOut | 淡出消失 |
| floatUp / floatDown | 向上/下飘散 |
| explode | 爆炸四散 |
| shrink | 缩小消失 |
| afterimage | 残影向右消散 |

## 常见问题

**Q：导出的 WebM 在某些软件中没有透明通道？**
A：请确认剪辑软件支持 VP9 Alpha。不支持的情况下请改用 MOV 格式。

**Q：MOV 导出卡住不动？**
A：首次导出需下载 ffmpeg.wasm（约 30MB），网络较慢时请等待进度条完成。

**Q：相同歌词每次生成的动画不一样？**
A：在「随机种子」输入框中填入固定字符串（如歌曲名），即可复现相同效果。

**Q：文字超出画面边界？**
A：布局引擎会自动缩小字号以适应画布，若仍有问题请检查单行歌词是否过长。
