# LrcCaptionGenerator

将 LRC 歌词文件转换为带动画效果的透明背景字幕视频，无需安装任何软件，直接在浏览器中运行。

## 功能特性

- 解析标准 LRC 歌词文件，自动同步字幕时间轴
- 12 种入场动画、3 种待机动画、7 种退场动画，随机组合，**每种特效参数均可在面板中调节**
- 支持「启用随机特效」开关：关闭时文字以无动效、居中固定样式呈现
- 相同随机种子可复现完全一致的动画效果
- 支持自定义背景（纯色或图片）、字体颜色与描边
- 逐行手动编辑：可单独覆盖每行的布局、特效及参数
- 实时预览，可同步播放音频
- 导出透明背景视频：WebM（VP9）或 MOV（ProRes 4444），或导出当前帧 PNG
- 纯前端，无需服务器，可作为静态页面部署

## 快速开始

### 环境要求

- Node.js 18+
- 现代浏览器（Chrome / Edge 推荐，需支持 MediaRecorder API）

### 安装与运行

```bash
cd LrcCaptionGenerator
npm install
npm run dev
```

浏览器访问 `http://localhost:5173` 即可使用。

## 使用说明

### 第一步：输入歌词

在左侧面板的文本框中粘贴 LRC 格式歌词，或点击「上传 LRC 文件」按钮导入文件。

LRC 格式示例：

```
[00:12.34]第一句歌词
[00:16.78]第二句歌词
[00:21.00]第三句歌词
```

### 第二步：配置参数

| 参数 | 说明 |
|------|------|
| **启用随机特效** | 勾选后显示随机种子输入框与骰子按钮；取消勾选则所有字幕以无动效、居中固定样式呈现 |
| **随机种子** | 控制动画随机性，相同种子产生相同效果；留空则每次随机 |
| **画布尺寸** | 1920×1080（横屏）、1080×1920（竖屏）、1080×1080（方形）、1280×720 |
| **背景** | 纯色或图片；图片可调节亮度、对比度、饱和度 |
| **字色 / 描边色** | 字体填充色与描边颜色 |
| **描边** | 描边宽度，0 为无描边 |

### 第三步：生成预览

点击「生成预览」按钮，右侧画布将显示动画效果。使用播放控件预览完整时间轴。

可选：上传音频文件以同步播放。

### 第四步（可选）：逐行编辑

生成预览后，右侧面板列出每一行歌词。点击任意行可展开编辑器，调整该行的：

- **布局**：位置、字号、字间距、对齐方式、旋转角度
- **特效**：入场 / 待机 / 退场效果，以及每种效果的专属参数（切换效果时参数区自动刷新）

点击「应用并预览」立即生效，「还原随机值」恢复系统随机结果。

### 第五步：导出

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| **PNG** | 导出当前帧静态图片 | 封面、缩略图 |
| **WebM** | VP9 编码，含透明通道，文件较小 | DaVinci Resolve、剪映桌面版等支持 VP9 Alpha 的软件 |
| **MOV** | ProRes 4444，含透明通道，兼容性最佳 | Final Cut Pro、Premiere、剪映手机版等 |

> MOV 导出首次使用时需加载约 30MB 的 ffmpeg.wasm，请耐心等待。

## 动画效果一览

### 入场（12 种）

| 效果 | 描述 | 可调参数 |
|------|------|----------|
| typewriter | 逐字打字机效果 | 出现时长、字符间隔 |
| slideLeft / slideRight | 从左/右侧滑入 | 滑动距离、动画时长、字符错位 |
| slideUp / slideDown | 从下/上方滑入 | 滑动距离、动画时长、字符错位 |
| scalePop | 弹性缩放弹出 | 动画时长、字符错位、弹性强度 |
| scatter | 从随机位置汇聚 | 水平/垂直散布范围、动画时长 |
| flipX | 水平翻转入场 | 动画时长、字符错位 |
| blurFade | 模糊渐显 | 模糊程度、动画时长、字符错位 |
| wave | 波浪式依次出现 | 波浪高度、动画时长、字符错位 |
| fadeIn | 纯透明度淡入 | 淡入时长、字符错位 |
| glitch | 故障闪烁定位 | 偏移强度、闪动次数、每次时长 |

### 待机（3 种）

| 效果 | 描述 | 可调参数 |
|------|------|----------|
| float | 整行上下浮动 | 浮动幅度、周期 |
| charJitter | 各字符微抖动 | 抖动幅度、速度 |
| breathe | 整行呼吸缩放 | 缩放量、周期 |

### 退场（7 种）

| 效果 | 描述 | 可调参数 |
|------|------|----------|
| fadeOut | 淡出消失 | 淡出时长 |
| floatUp / floatDown | 向上/下飘散 | 飘动距离、动画时长、字符错位 |
| explode | 爆炸四散 | 水平/垂直散布范围、动画时长 |
| shrink | 缩小消失 | 动画时长、字符错位 |
| afterimage | 残影向右消散 | 残影距离、持续时长 |
| blurOut | 模糊淡出 | 模糊程度、动画时长、字符错位 |

## 常见问题

**Q：导出的 WebM 在某些软件中没有透明通道？**
A：请确认剪辑软件支持 VP9 Alpha。不支持时请改用 MOV 格式。

**Q：MOV 导出卡住不动？**
A：首次导出需下载 ffmpeg.wasm（约 30MB），网络较慢时请等待进度条完成。

**Q：相同歌词每次生成的动画不一样？**
A：在「随机种子」输入框中填入固定字符串（如歌曲名），即可复现相同效果。

**Q：文字超出画面边界？**
A：布局引擎会自动缩小字号以适应画布，若仍有问题请检查单行歌词是否过长。

**Q：我不想要任何动效，只想让字幕静静出现？**
A：取消勾选「启用随机特效」，所有字幕将以简单淡入淡出、居中固定在画面底部的方式呈现。
